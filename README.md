# Flutter апликација за рецепти 

Апликација за прикажување на рецепти, развиена во Flutter, која им овозможува на корисниците интуитивен и лесен пристап до широк спектар на јадења.
На почетниот екран на апликацијата се прикажуваат рецепти од различни категории, при што за секоја категорија се прикажува име, слика и краток опис. При избор на категорија, корисникот добива преглед на сите рецепти во рамки на grid layout со нивните имиња и слики, а изборот на поединечен рецепт отвора детален приказ со инструкции за подготовка, листа на состојки, илустративна слика и YouTube видео за рецептот. Дополнително, апликацијата нуди функција за прикажување на случаен рецепт на денот, кој служи како брз извор на инспирација за подготовка на нови јадења.
